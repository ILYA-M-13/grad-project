<h1 align="center">Блоговый движок</h1>
Это проект "Блоговый движок", позволяющий писать свои посты и комментировать посты других людей. Предусмотрены функции аутентификации, модерирования, рефакторинга постов, ведения статистики: лайков, дизлайков, просмотров.
<h4 align="left">Настройка</h4>


Создайте пустую базу данных. В application.yml укажите ее URL, username, password.

<h3 align="left">Используемые технологии</h3>

- Java 11
- Spring Boot 2.3.12
- Maven
- MySQL 8
- Hibernate 5.5.4
