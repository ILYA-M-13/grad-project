<h1 align="center">Блоговый движок</h1>

##Это учебный проект "Блоговый движок", позволяющий писать свои посты и комментировать посты других людей. Предусмотрены функции аутентификации, модерирования, рефакторинга постов, ведения статистики: лайков, дизлайков, просмотров.

### Настройка

Создайте пустую базу данных. В application.yml укажите ее URL, username, password.

###Используемые технологии

- Java 11
- Spring Boot 2.3.12
- Maven
- MySQL 8
- Hibernate 5.5.4
